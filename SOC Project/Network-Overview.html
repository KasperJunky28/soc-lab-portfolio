Network Architecture & Segmentation

Overview

The lab environment is designed to simulate a segmented enterprise network, separating internal infrastructure from external attacker access. This segmentation enables realistic simulation of intrusion scenarios, lateral movement, and detection workflows.

The architecture combines identity services, enterprise systems, and centralized monitoring to reflect a real-world organizational environment.

Network Design

  • Internal Network (192.168.59.0/24): Corporate infrastructure and endpoints
  • Attacker Network (192.168.100.0/24): External threat simulation environment
  • SIEM Dual-Homed Configuration: Connected to both networks for log collection and visibility

The separation between internal and attacker networks allows controlled simulation of external attacks while maintaining visibility across both environments.

IP Addressing & Key Systems

  • Domain Controller: 192.168.59.10
  • File Server: 192.168.59.20
  • Web server: 192.168.59.30
  • SIEM Server: 192.168.59.40 (internal) / 192.168.100.1 (attacker-facing)
  • Endpoints: DHCP-assigned within internal network
  • Attacker Machine:192.168.100.10 (Located within attacker network segment)

Network Diagram

Traffic Flow & Visibility

  • Endpoints generate authentication and activity logs (Windows Event Logs, Sysmon)
  • Logs are forwarded to the SIEM server for analysis and correlation
  • Network traffic is monitored by Suricata IDS for anomaly detection
  • Attacker traffic originates from the external network and targets internal systems

This setup enables full visibility across endpoint, network, and attacker activity, supporting effective detection and investigation workflows.

Security Considerations

Network segmentation is a critical control that limits attack surface and enables detection. By isolating attacker infrastructure from internal systems, the lab accurately models real-world threat scenarios such as initial access and lateral movement.

The dual-homed SIEM architecture ensures visibility across both segments, allowing correlation between external attack activity and internal system behavior.
