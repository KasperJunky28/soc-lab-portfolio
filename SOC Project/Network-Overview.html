Network Overview

Network Architecture

  • Internal Network: 192.168.59.0/24
  • Domain Controller: 192.168.59.10
  • SIEM Server: 192.168.59.40 / 192.168.100.1
  • Clients: DHCP range
  • Attacker Network: 192.168.100.0/24

The lab is divided into internal corporate and attacker networks, allowing realistic simulation of lateral movement and external threats.

Network Diagram

Key Components

  • Domain Controller: Identity, DNS, DHCP, NTP
  • SIEM Server: Log collection and detection
  • Endpoints: User activity simulation
  • Attacker Machine: Offensive testing

Why This Matters

Understanding network architecture is critical for detecting anomalies, identifying lateral movement, and analyzing attack patterns.
