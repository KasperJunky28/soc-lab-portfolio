SOC Lab Project

Project Overview

This project demonstrates the design and implementation of a complete Security Operations Center (SOC) lab environment.

The lab simulates a real enterprise network including Active Directory, endpoints, SIEM monitoring, IDS, and attacker simulation.

Objectives

  • Build a realistic enterprise network environment
  • Deploy centralized monitoring using SIEM (Wazuh)
  • Implement endpoint and network security tools
  • Simulate attacks and analyze detection capabilities
  • Map detections to MITRE ATT&CK framework

Technologies Used

  • Windows Server (Active Directory, DNS, DHCP)
  • Windows 10 Clients
  • Ubuntu Server (Wazuh SIEM)
  • Suricata IDS
  • Sysmon
  • Kali Linux (Attacker Machine)

Architecture Summary

The environment consists of multiple virtual machines connected through internal and NAT networks, simulating corporate and attacker segments.

  • Domain Controller managing identity and services
  • Client machines generating activity and logs
  • SIEM server collecting and analyzing logs
  • Attacker machine simulating real-world threats
