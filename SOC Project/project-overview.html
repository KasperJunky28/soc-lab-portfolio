SOC Lab Project

Project Overview

This project demonstrates the design, implementation, and security analysis of a full Security Operations Center (SOC) lab environment.

The lab simulates a real-world enterprise network and focuses on the complete attack lifecycle — from external reconnaissance to internal compromise and data exfiltration — as well as detection, analysis, and security improvement.

The project emphasizes both offensive simulation and defensive analysis, highlighting detection capabilities, visibility gaps, and practical security hardening.

Objectives

  • Design and deploy a realistic enterprise network environment
  • Implement centralized monitoring and detection using Wazuh SIEM
  • Simulate real-world attack scenarios (external and internal)
  • Analyze detection coverage and identify security gaps
  • Map attack techniques to the MITRE ATT&CK framework
  • Implement security hardening based on identified weaknesses

Technologies Used

  • Windows Server (Active Directory, DNS, DHCP)
  • Windows 10 Clients
  • Apache Web Server
  • SMB / Windows File Server
  • Ubuntu Server (Wazuh SIEM)
  • Suricata IDS
  • Sysmon
  • Kali Linux (Attacker Machine)

Architecture Summary

The lab environment is composed of multiple interconnected virtual machines, segmented to simulate both internal enterprise infrastructure and external attacker access.

  • Domain Controller (Active Directory, DNS, DHCP)
  • File Server with shared resources and permission structures
  • Web Server hosting internal applications
  • SIEM Server (Wazuh) for centralized logging and detection
  • Suricata IDS for network-level monitoring
  • Windows endpoints generating user activity
  • Kali Linux attacker machine simulating external threats
