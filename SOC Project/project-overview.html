End-to-End Security Operations Case Study

Project Overview

This project presents a full-scale Security Operations Center (SOC) environment designed to simulate a realistic enterprise network and evaluate security posture across the complete attack lifecycle.

The focus is not only on executing attacks, but on analyzing detection capabilities, identifying visibility gaps, and implementing measurable security improvements.

Project Objectives

  • Design and deploy a realistic enterprise network architecture
  • Implement centralized monitoring and detection using Wazuh SIEM
  • Simulate external and internal attack scenarios
  • Identify detection gaps and improve visibility
  • Map activity to the MITRE ATT&CK framework
  • Apply and validate security hardening strategies

Architecture & Technology Overview

The environment consists of multiple interconnected systems representing a segmented enterprise network, combining identity infrastructure, enterprise services, and security monitoring components.

  • Domain Controller: Active Directory, authentication, DNS, DHCP
  • File Server: Sensitive data storage with structured access control
  • Web Server: Internal service hosting
  • Windows Endpoints: User activity simulation
  • Wazuh SIEM: Centralized logging, correlation, and alerting
  • Suricata IDS: Network traffic monitoring
  • Sysmon: Endpoint-level visibility
  • Kali Linux: Adversary simulation

Threat Model

  • External Threats: Reconnaissance, scanning, initial access attempts
  • Internal Threats: Privilege misuse, lateral movement, data access
  • Objective: Simulate unauthorized access and data exfiltration

Detection Strategy

  • Endpoint telemetry via Sysmon
  • Windows Event Logs (4624 – Login, 4663 – File Access)
  • Network monitoring with Suricata IDS
  • SIEM correlation using Wazuh

Key detection logic includes correlating successful logins with file access activity to identify potential data exfiltration behavior.

Key Skills Demonstrated

  • Enterprise Network Design & Deployment: Segmented lab architecture, VM networking, and infrastructure planning
  • Active Directory & Identity Management: Domain setup, user management, authentication policies
  • Enterprise Service Configuration: File Server (SMB permissions), Web Server (Apache), DNS & DHCP services
  • SIEM Monitoring & Detection Engineering: Log collection, rule creation, and correlation (Wazuh)
  • Endpoint & Network Visibility: Sysmon telemetry and Suricata IDS analysis
  • Adversary Simulation: External and internal attack scenarios using Kali Linux
  • MITRE ATT&CK Mapping: Mapping attack techniques to detection logic
  • Security Gap Analysis: Identifying visibility and control weaknesses
  • Security Hardening & Validation: Implementing improvements and validating via attack replay
